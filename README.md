# Rust-OpenSMTPD

Rust binding for [OpenSMTPD] filters.


# Features

This is a work in progress, the API is **not** stabilized yet.

- [ ] Thread-pool
- [x] Reports
- [ ] Filters
- [x] Session-level context
- [ ] Pool-level context
- [ ] Filter-level context


[OpenSMTPD]: https://www.opensmtpd.org/
